# Tic-Tac-Toe
play some tic-tac-toe against a bot or a person.

By default you'll play first against a bot.

## Install
* Ruby 2.4+
* `bundle`

## Usage
`ruby tictactoe.rb`

Add `--1v1` for local multiplayer

`ruby tictactoe.rb --1v1`
